TinTint 網站改版

TinTint 2017 Web Design

「點點印」網站重新設計，以主流程的三個頁面優先規劃：首頁 / 單一產品頁 / 產品系列頁。
為迎合使用者螢幕越來越大，以及配合多種裝置的需求，希望逐步將舊有框架：頁寬 980px 改為滿版，
原本電腦版與手機版分開設計、維護的架構，也逐步調整成 RWD。

  • Mockup
  • RWD 與 F2E 合作進行
  • CSS 調整

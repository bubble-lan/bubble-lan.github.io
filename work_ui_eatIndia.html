Eat India!

Side Project 2018 UI Design
Landing Page (PC) / App / 以 Sketch 繪製

去印度自助旅行，有許多方便的 App / 平台是背包客的好朋友。
大家都知道印度火車超會誤點， India Rails 可以即時查詢火車目前的行駛狀態；
Cleartrip、MakeＭyTrip 結合了訂購機票、住宿和鐵路等服務。
要尋找當地美食，除了看看 IG 上的網美在哪裡出沒，不可或缺的還是一本《Lonely Planet》，

實際走訪印度的經驗，在找尋書上介紹的餐廳時，會發現 Google Map 定位和書上的地址不同，
知名餐廳 / 小店經常出現山寨版，在迷宮一般的印度小巷裡，相同店名的指示牌，一個指東，一個指西，誰才是正版？完全迷惑。
雖然有 Trip Advisor 可以查看餐廳評價，但街邊小吃看起來好誘人，想要來一杯熱呼呼的印度奶茶，又不知道腸胃是否夠強壯。
希望能有一款專屬印度的就餐搜尋指南，讓想好好吃一頓飯又怕踩雷的旅人，不再團團轉，又能吃得安心（應該啦）。

項目 Project

Eat India!「印度吃什麼」

  • 網站 Landing Page (PC)
  • App UI 設計

前言 Introduction

針對到印度自助旅行的使用者，提供有搜尋評價及地圖功能的服務，類似全球性的 Zomato 或中國的大眾點評、台灣的愛評網。

使用者故事 User Story

  • 使用者可以從平台的分類或推薦獲取靈感
  • 使用者可以城市或關鍵字快速搜尋
  • 使用者可以定位目前的位置，查看附近店家
  • 使用者可以對店家給予評價或留言回饋
  • 使用者可以收藏店家，加入口袋名單

Catbnb

Side Project 2018 UI Design

身邊許多養貓的朋友，總在旅行、出差或返鄉時，有貓咪寄宿的需求。
因此以貓咪版的 Airbnb App 作為發想，希望貓奴們在寵物旅館之外，有更多的選擇，並且提供更簡便的平台。

  • UI 設計練習
  • 參考平台 Airbnb 及 Booking.com
  • 以 Sketch 設計
